## Cara pengerjaan

Clone repository

```bash
  git clone https://github.com/iqbalherdianysah/vrent-frontend.git
```

Checkout ker branch masing-masing

```bash
  git checkout -b nama_branch origin/nama_branch
```

Install dependencies

```bash
  npm install
```

Jalankan server

```bash
  npm run dev
```
